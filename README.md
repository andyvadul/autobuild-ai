# Autobuild.ai

> **Your Spec, AI's Creation** — Build production-ready applications by writing specifications while AI handles all the code

---

## What is Autobuild.ai?

A free, open-source course that teaches business professionals to build real applications without writing code. You focus on **what** to build (features, business logic, user experience), while AI agents handle **how** to build it (code, databases, APIs, deployment).

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   YOUR INPUT:              YOUR OUTPUT:                         │
│   Spec Template    ───►    Production App                       │
│   (business terms)         (deployed & working)                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Who Is This For?

- **Business Operators** — Build internal tools and automate workflows
- **Founders** — Create MVPs without hiring developers
- **Product Managers** — Prototype ideas and reduce dev dependency
- **Citizen Developers** — Level up from no-code to production apps

**No programming experience required.**
